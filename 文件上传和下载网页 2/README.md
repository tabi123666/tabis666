# 文件共享系统

一个基于Node.js和Express的多用户文件共享系统，支持文件上传、搜索和下载功能。

## 功能特性

- ✅ **多用户共享**：任何人都可以上传文件，所有人都能通过搜索下载
- ✅ **文件上传**：支持多种文件类型（Word、PDF、图片）
- ✅ **智能搜索**：支持按文件名、文件类型、扩展名等搜索
- ✅ **拖拽上传**：支持通过拖拽方式快速上传文件
- ✅ **响应式设计**：适配各种设备屏幕
- ✅ **实时反馈**：上传和下载过程提供实时状态提示

## 技术栈

- **前端**：HTML5、CSS3、JavaScript
- **后端**：Node.js、Express
- **文件上传**：Multer
- **数据库**：SQLite3
- **跨域支持**：CORS

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动服务器

```bash
node server.js
```

### 3. 访问系统

打开浏览器，访问：http://localhost:3000

## 使用说明

### 上传文件

1. 点击"选择文件"按钮或拖拽文件到上传区域
2. 选择要上传的文件（支持多选）
3. 系统会自动验证文件类型并上传
4. 上传完成后会显示成功提示

### 搜索文件

1. 在搜索框中输入关键词
2. 系统会实时显示匹配的文件
3. 支持搜索：
   - 文件名
   - 文件类型（如：pdf、word、image）
   - 中文类型（如：文档、图片）
   - 文件扩展名

### 下载文件

1. 在搜索结果中找到要下载的文件
2. 点击"下载"按钮即可下载

## 支持的文件类型

- **文档**：.doc, .docx
- **PDF**：.pdf
- **图片**：.jpg, .jpeg, .png, .gif
