# 文件上传和下载系统

一个支持中文文件名的文件上传、下载、搜索和删除的网页应用。

## 功能特性

- ✅ 支持中文文件名上传
- ✅ 支持中文文件名下载
- ✅ 文件搜索功能
- ✅ 文件删除功能
- ✅ 空状态提示
- ✅ 响应式设计

## 技术栈

- **前端**：HTML、CSS、JavaScript
- **后端**：Node.js、Express
- **数据库**：SQLite
- **文件上传**：Multer

## 部署说明

### 本地运行

```bash
npm install
npm start
```

访问：http://localhost:3000

### 云部署

1. 创建GitHub仓库
2. 推送代码到GitHub
3. 在Render等平台部署

## 使用说明

1. 点击"选择文件"按钮上传文件
2. 使用搜索框搜索文件
3. 点击"下载"按钮下载文件
4. 点击"删除"按钮删除文件
